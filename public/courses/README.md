# Carpeta de Imágenes de Cursos

Coloca aquí las imágenes de portada (thumbnails) de los cursos.

## Cómo usar:

1. Coloca tus imágenes aquí (formato JPG, PNG o WebP)
2. En `src/data/courses.js`, cambia la ruta:
   ```javascript
   thumbnail: "/courses/nombre-archivo.jpg"
   ```

## Ejemplo:
- Archivo: `fundamentos-medicina.jpg`
- En el código: `thumbnail: "/courses/fundamentos-medicina.jpg"`

## Tamaño recomendado:
- 800x500px o más
- Formato horizontal (16:9) funciona mejor
