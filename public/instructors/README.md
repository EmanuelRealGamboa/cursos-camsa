# Carpeta de Imágenes de Instructores

Coloca aquí las imágenes de los instructores.

## Cómo usar:

1. Coloca tus imágenes aquí (formato JPG, PNG o WebP)
2. En `src/data/instructors.js`, cambia la ruta:
   ```javascript
   image: "/instructors/nombre-archivo.jpg"
   ```

## Ejemplo:
- Archivo: `maria-garcia.jpg`
- En el código: `image: "/instructors/maria-garcia.jpg"`

## Tamaño recomendado:
- 400x400px o más
- Formato cuadrado funciona mejor
